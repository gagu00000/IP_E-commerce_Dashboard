## ğŸ¯ Overview

This executive analytics dashboard provides comprehensive insights for a luxury e-commerce platform operating in the UAE market. Built with a premium aesthetic featuring gold accents and dark themes, it reflects the high-end positioning of the brand while delivering actionable business intelligence.

### Key Highlights

- **Real-time KPI Monitoring**: Track 14+ key performance indicators
- **Customer Intelligence**: Deep dive into CLV by segment (VIP, Premium, Regular)

- **Revenue Analytics**: Multi-dimensional revenue analysis by channel, city, and category
